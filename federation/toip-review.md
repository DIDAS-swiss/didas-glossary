# ToIP Main Glossary — Federation Review

Term-by-term review of the [Trust over IP Main Glossary](https://glossary.trustoverip.org/)
([source repo](https://github.com/trustoverip/ctwg-main-glossary)) for federation into the
DIDAS Glossary. This table is the review state: update rows in the same pull request that
implements a decision.

## How to review

1. Pick a batch of `pending` terms (10–20).
2. For each, decide per the three-tier rule in [GOVERNANCE.md](../GOVERNANCE.md):
   - **adopt** (Tier 1) — ToIP definition is sufficient: add a one-line
     `[[tref: toip-glossary, <term>, <DE>, <FR>, <IT>]]` term file in the same PR
   - **extend** (Tier 2) — needs Swiss/swiyu context: open a term-proposal issue
   - **own** (Tier 3) — Swiss law or DIDAS-specific meaning prevails: open a term-proposal issue
   - **skip** — not relevant to the DIDAS/swiyu scope
3. Fill in `Decision`, `DE/FR/IT aliases` (for adopt), `Reviewer`, and the local file link.
4. Submit the batch as one PR; editorial approval merges it.
